# Lunari Kinematic Character 2D

This demo remakes Godot's official 2D kinematic character example with Lunari
scripts. It keeps the original scene layout and assets while replacing the
gameplay logic with `.lu` files.

Language: Lunari

Renderer: Compatibility

Original demo source: https://github.com/godotengine/godot-demo-projects/tree/master/2d/kinematic_character
